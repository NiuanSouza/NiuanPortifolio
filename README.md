# Niuan Portfolio

Bem-vindo ao meu portfólio profissional! Este projeto foi desenvolvido para demonstrar minhas habilidades em desenvolvimento frontend, utilizando tecnologias modernas e focando em uma experiência de usuário responsiva e dinâmica.

## 🚀 Acesso ao Projeto
Você pode visualizar o projeto em funcionamento através do link abaixo:
[https://niuanportifolio.onrender.com](https://niuanportifolio.onrender.com)

## 🛠 Tecnologias Utilizadas
- **React**: Biblioteca principal para a interface do usuário.
- **TypeScript**: Para maior segurança e robustez no código.
- **Vite**: Ferramenta de build rápida e eficiente.
- **CSS Modules**: Para estilização isolada e modularizada de cada componente.

## ✨ Funcionalidades
- **Sistema de Temas**: O site conta com um alternador de temas dinâmico, permitindo uma experiência visual personalizada.
- **Arquitetura Escalável**: Estrutura organizada em componentes, facilitando a manutenção e a adição de novos recursos.
- **Favicon Dinâmico**: O favicon do site se ajusta automaticamente de acordo com o tema selecionado.

## 🔮 Planos Futuros
Estou sempre buscando evoluir meus projetos. Para as próximas atualizações, planejo desenvolver uma **API dedicada** que irá consultar a loja oficial do VS Code em tempo real. O objetivo é permitir que os usuários possam pesquisar, visualizar e organizar temas diretamente no meu portfólio de forma automatizada.
